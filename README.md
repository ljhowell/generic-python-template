# Generic Python Template

## Overview
This is a generic Python template repository for new projects.

Includes:
- `src/` folder for source code
- `tests/` for tests
- Dockerfile for docker containers
- GitHub Actions workflow for CI/CD
- Pre-commit configuration for linting and cleanup

## How to Use
1. Clone the repository or use it as a GitHub template.
2. Add your code to `src/`.
3. Add tests to `tests/`.
4. Update `Dockerfile` CMD for your project.
